# redemrp_doorlocks
RedM door locks made for redem_roleplay

# Installation
1. Clone this repository.
2. Extract the zip.
3. Put redemrp_doorlocks to your resource folder.
4. Add "start redemrp_doorlocks" in your "server.cfg".
5. Profit

# Required resource
- redem_roleplay
